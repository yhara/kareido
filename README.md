# Kareido

Yet another LLVM example program, inspired by [the official one](http://llvm.org/docs/tutorial/)

## The Language

Number 

	12345
	1.3

String

	"hello"

Unary operators

	-1
	+1

Binary operators

	1 + 2
	1 - 2
	1 * 2
	1 / 2   //=> 0.5

Conditional

	if (x == 1) {   // curly bracket is mandatory
		y = 2
	}
	else {
		y = 3
	}

Loop

	for (i = 0, y < 3, +1) {
		x *= 5
	}

Var
	
	var (a = 1, b = 2) {
	}
